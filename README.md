 An aircrack-ng script built with Go

# Installation and running
1. Download and install the [Go compiler](https://go.dev)  
   
   Or use a package manager:  
   ```
   sudo apt install golang
   ```  
   ```
   sudo pacman -S go
   ```
2. To directly run the script:  
   ```
   go run crack.go
   ```
3. To build and run:  
   
   ```
   go build .
   ./crackongo
   ```
